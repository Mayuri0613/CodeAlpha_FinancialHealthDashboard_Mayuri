# Financial Health Dashboard – CodeAlpha Internship

## Project Overview
This project is developed as part of the CodeAlpha Power BI Internship.

The dashboard provides a complete financial analysis of an organization using interactive visualizations and forecasting techniques.

## Features
- Total Sales, Profit, and Units Sold KPIs
- Profit Trend Analysis
- Forecasting for Financial Planning
- Country-wise Sales Analysis
- Product-wise Profit Distribution
- Interactive Slicers and Filters

## Tools & Technologies
- Power BI Desktop
- Microsoft Excel
- GitHub

## Files Included
- Power BI Dashboard (.pbix)
- Dataset (.xlsx)
- Dashboard Screenshots

## Dashboard Insights
- Identified high-performing countries based on sales.
- Analyzed profitability trends over time.
- Forecasted future financial growth.
- Evaluated product performance contribution.
